Assumptions

:

SEM Imaging - In order to obtain the pore sizes we used in the calculator, SEM imaging of the filter layer with a magnification between 1.4kx and 1.5kx were taken and subject to analysis using Fiji and the DiamaterJ plugin

Fiji 1.51w - Fiji is a version of ImageJ - an NIH developed image processing program - and to maintain accuracy and reproducibility, please use the version of Fiji (1.51w) that was used to analyze SEM images of the PPE material

DiameterJ - DiameterJ is a plugin that is used through Fiji and ImageJ. DiameterJ can be split into two plugins - DiameterJ Segment and DiameterJ 1-018. DiameterJ Segment turns SEM images into Binary black/white imaging according to specific algorithms. DiameterJ 1-018 is used to measure pore sizes and fiber characteristics using the output from DiameterJ Segment

ASTM Standards - Several standards were followed in order to create reproducibility and replicability
